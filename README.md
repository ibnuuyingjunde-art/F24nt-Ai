# F24nT AI

## 🚀 AI Chat Platform with Tools & Terminal

F24nT AI adalah platform chat AI yang powerful dengan dukungan:
- 18+ Model AI (Claude, GPT, DeepSeek, Grok, Llama, Gemini)
- Terminal untuk menjalankan kode (Python, JS, HTML, Bash)
- Upload & analisis file (gambar, video, dokumen, kode)
- Google Image Search
- NekoPoi (Abyz Plan)
- Memory & conversation history
- Incognito mode
- Premium Plan (Abyz & Gokil)
