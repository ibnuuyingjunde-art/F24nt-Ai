## 📋 Deskripsi

F24nT AI adalah platform chat AI all-in-one dengan dukungan:
- **18+ Model AI** (Claude, GPT, DeepSeek, Grok, Llama, Gemini)
- **Terminal** untuk menjalankan kode (Python, JS, HTML, Bash)
- **Upload & Analisis** file (gambar, video, dokumen, kode)
- **Google Image Search**
- **NekoPoi** (Abyz Plan)
- **Memory & Conversation History**
- **Incognito Mode**
- **Dark/Light Theme**
